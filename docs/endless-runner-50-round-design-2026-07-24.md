# 안엘런 50라운드 난이도·패턴 설계 (2026-07-24)

## 적용 원칙

다른 러너 게임의 맵, 캐릭터, 그래픽, 음원, 고유 명칭과 장애물 배치를 복제하지 않는다.
조사에서 확인한 **학습 순서, 난이도 구간, 안전 행동 안내, 고난도 뒤 회복 구간**만 안엘런의
기존 초원 에셋과 점프·2단 점프·슬라이드 조작으로 재해석한다.

참고한 구조는 다음과 같다.

- CookieRun: OvenBreak 공식 소개는 젤리를 먹으며 점프와 슬라이드로 장애물을 피하는 간단한
  조작을 핵심으로 설명한다.
  - https://www.cookierun.com/presskit/
- Cookie Run 커뮤니티 위키의 장애물 정리는 낮은 장애물·높은 장애물·매달린 장애물·비행
  장애물을 각각 1단 점프·2단 점프·슬라이드 등으로 구분하고, 젤리 배열이 다가오는 안전
  경로를 안내한다고 설명한다.
  - https://cookierun.fandom.com/wiki/Obstacles
- Main Story 난이도는 Calm·Normal·Hard·Master처럼 속도와 충돌 페널티를 구간 단위로
  올린다. 한 수치만 계속 올리기보다 명확한 난이도 묶음을 보여주는 참고로 사용했다.
  - https://cookierun.wiki/w/Main_Story
- Escape from the Oven의 스테이지 설명에는 새떼·얇은 발판·구덩이·움직이는 바닥을
  섞되, 어려운 구간 사이에 안전한 평지와 보상 구간을 두는 사례가 기록돼 있다.
  - https://cookierun.wiki/w/Escape_from_the_Oven_%28OvenBreak%29
- Subway Surfers의 Mystery Hurdles 공식 도움말은 한 주행 안에서 네 난이도 구간을 올리고
  단계 상승을 화면에 알리며, 점프·구르기·선택형 방해물을 구분한다고 설명한다.
  - https://sybo.helpshift.com/hc/en/5-subway-surfers/faq/139-mystery-hurdles/
- 절차 생성 러너 연구는 무작위 조합이 막힌 길이나 기술적으로 불가능한 구간을 만들 수
  있으므로 생성과 함께 플레이 가능성·다양성·제어 가능성을 검증해야 한다고 지적한다.
  - https://arxiv.org/abs/2605.01783

## 난이도 곡선

- 라운드당 15초, 50라운드는 시작 후 12분 15초에 진입한다.
- 실제 라운드와 HUD는 `50/50`에서 멈추고 이후 최고 난이도를 유지한다.
- 1~50 속도는 약 `245px/s → 505px/s`로 완만하게 오르고, 라운드 내부 가속을 포함한
  최종 상한은 약 `526px/s`다.
- 첫 학습 라운드의 최소 생성 간격은 `464px`이며 이후 점진적으로 줄되 `300px` 아래로
  내려가지 않는다.
- 15·25·35·45라운드는 직전보다 속도를 낮추고 간격을 넓힌 회복 구간이다.
- 각 라운드 첫 코스는 고정해 새 패턴이 난수 때문에 누락되지 않게 한다. 이후에는 해당
  라운드 전용 패턴 풀과 일반 장애물을 섞는다.
- 연속 패턴이 차지한 전체 길이를 생성 쿨다운에 반영해, 마지막 장애물과 다음 무작위
  장애물 사이에도 최소 안전 간격을 보장한다.

## 코스 패턴 계약

- `single-jump`: 낮은 지형 하나
- `single-slide`: 줄에 매달린 뱀 하나
- `ground-gap`: 단독 구덩이
- `jump-pair`: 착지 시간을 둔 낮은 장애물 두 개
- `jump-slide`: 점프 체공시간과 슬라이드 진입시간을 합친 간격 보장
- `slide-jump`: 슬라이드를 점프로 즉시 취소할 수 있는 간격 보장
- `creature-wave`: 같은 슬라이드 통로를 쓰는 벌·파랑새 연속
- `platform-pair`: 고정 발판 뒤 상하 이동 발판, 각각 바로 아래에 구덩이
- `mixed-relay`: 1단 점프 → 슬라이드 → 2단 점프 바위탑

코스 내부 간격은 픽셀 상수가 아니라 해당 라운드 속도에 필요한 반응시간을 곱해 계산한다.
50라운드에서도 점프→슬라이드는 약 `0.94초`, 슬라이드→점프는 약 `0.58초`, 발판 전환은
약 `0.9초`를 보장한다.
