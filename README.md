# run-hoban-run

검단호수공원역 호반써밋 경마 추첨 웹게임입니다.

## 실행

```bash
npm run start
```

브라우저에서 `http://localhost:5174`를 엽니다.

## 테스트

```bash
npm test
```

Docker로 테스트:

```bash
npm run test:docker
```

## 규칙

- 제목: 달려라 검단호수공원 호반써밋
- 참가자: 최대 800명
- 트랙: 3가지 유형
- 경기 시간: 60초
- 장애물: 무작위 10회
- 장애물 통과 확률: 참가자별 49%
- 스킬: 참가자별 5% 확률로 10초간 발동
- 통과: 운영자가 지정한 순위 범위 안에 들어온 참가자
