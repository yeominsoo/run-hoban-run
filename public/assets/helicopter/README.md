# Helicopter Asset

- Direct source URL: https://pub-8b7609252a6742d7b029292aaede7f7f.r2.dev/3d-models/helicopter.glb
- Notes: FreePixel describes the 3D assets as GLB files with no attribution required, free for commercial and personal projects. The game applies its own military material pass and hardpoint details at runtime.
- Optimization: Reduced with glTF Transform using mesh simplification, KHR_mesh_quantization, and 1024px texture resizing so the first page load does not need the original 5.9MB model.
