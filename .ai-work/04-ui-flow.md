## 화면 구성

앱은 3개 영역으로 구성됩니다.

| 영역 | 역할 |
| --- | --- |
| 상단 조작 바 | 설정, 결과, 규칙 갱신, 레이스 시작, 새 레이스, CSV |
| 중앙 레이스 화면 | 가로로 긴 60초 레이스 진행 표시 |
| 설정 레이어 | 참가자와 경기 조건 입력 |
| 결과 레이어 | 실시간 순위와 최종 결과 표시 |

## 운영 패널

운영 패널에서 설정하는 값:

- 참가자 목록
- 샘플 참가자 생성: 80명, 180명, 800명
- 트랙 선택
- 통과 시작 순위
- 통과 종료 순위
- 재생 속도

시드는 화면에 노출하지 않고 내부 값으로만 유지합니다.

주요 버튼:

- `설정`: 참가자와 규칙 입력 레이어를 엽니다.
- `결과`: 순위와 통과자 레이어를 엽니다.
- `규칙 갱신`: 현재 입력값으로 레이스 결과를 미리 생성합니다.
- `레이스 시작`: 생성된 레이스를 화면에 재생합니다.
- `새 레이스`: 새로운 내부 시드로 레이스를 다시 생성합니다.
- `CSV`: 전체 최종 순위를 CSV로 내보냅니다.

## 레이스 화면

레이스 화면에서 표시하는 값:

- 현재 트랙명
- 60초 진행률
- 호수, 나무 라인, 관중석, 출발선, 결승선을 포함한 그래픽 트랙
- 장애물 위치와 통과된 장애물 표시
- 상위권 레이스 라인
- 참가자의 상태
- 이름이 왼쪽, 말 이미지가 오른쪽에 있는 움직이는 참가자 표시
- 참가자별 고유 색상 표시

참가자 상태:

- 일반 주행
- 스킬 발동
- 장애물 실패 후 감속

## 결과 패널

결과 패널에서 표시하는 값:

- 참가자 수
- 통과자 수
- 스킬 발동 수
- 장애물 통과/실패 총계
- 실시간 상위 10명
- 최종 통과자 목록
- 최종 상위 40명 표

## 운영 순서

1. 참가자 목록을 붙여넣거나 샘플 버튼을 누릅니다.
2. 트랙을 선택합니다.
3. 통과 시작 순위와 종료 순위를 입력합니다.
4. `규칙 갱신`으로 결과를 준비합니다.
5. `레이스 시작`으로 화면 재생을 시작합니다.
6. 레이스 종료 후 통과자 목록과 최종 순위를 확인합니다.
7. 필요하면 `CSV`로 전체 결과를 저장합니다.
